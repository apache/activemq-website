ActiveMQ Artemis 1.5.1 Release Notes

A complete list of JIRAs for the 1.5.1 release can be found at the Apache ActiveMQ Artemis project JIRA

Bug Fixes:

  • [ARTEMIS-748] - AddressSize show a negative number.
  • [ARTEMIS-823] - Broker doesn't settle rejected messages
  • [ARTEMIS-827] - AMQP test client validator classes not thread safe
  • [ARTEMIS-840] - Passwords logged in plaintext
  • [ARTEMIS-842] - JMSMessageID doesn't appear to work in selector
  • [ARTEMIS-849] - CDI Integration doc missing
  • [ARTEMIS-852] - Add indices to tables from JDBC backend to increase performance
  • [ARTEMIS-859] - BACKLOG_PROP_NAME is not part of allowable properties on the acceptor
  • [ARTEMIS-860] - openwire protocol not installed in artemis-osgi

Improvements:

  • [ARTEMIS-814] - Support outgoing connections for AMQP
  • [ARTEMIS-841] - Hash Processors lock free lazy singleton instantiation
  • [ARTEMIS-845] - Making Default Acceptor Protocols a bit more clear
  • [ARTEMIS-846] - Once the disk is full the blocked producer should log about disk full
  • [ARTEMIS-863] - The broker should deal with Network Failures.

Tasks:

  • [ARTEMIS-865] - Remove e.printStackTrace() from code
