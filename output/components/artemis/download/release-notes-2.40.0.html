ActiveMQ Artemis 2.40.0 Release Notes

  • A list of commits.

  • Please see the “Versions” chapter in the User Guide for upgrade instructions.

Note: This release requires use of Java 17 or above.

  • ARTEMIS-5133 - FB.UWF_UNWRITTEN_FIELD in ArtemisClientPlugin.java

  • ARTEMIS-5182 - STOMP subscriber w/noLocal + selector missing messages
