New and Noteworthy

This release adds some great new features to the NMS ActiveMQ library and fixes several bugs found in the 1.3.0 release.Â  Some highlights for this release.

  • Improved URI handling in all NMS clients.
  • Improved handling of Request Timeout settings.

  • Added a new MessageTransformer API.
