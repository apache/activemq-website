Apache.NMS.ActiveMQ

The ActiveMQ NMS client is a .NET client that communicates with the ActiveMQ broker using its native Openwire protocol. This client supports advanced features such as Failover, Discovery, SSL, and Message Compression. This section of the NMS site provides documentation specific to the NMS.ActiveMQ client, for documentation on the other NMS clients refer to the Main NMS page.
