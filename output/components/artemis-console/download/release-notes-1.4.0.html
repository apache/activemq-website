Bugs Fixed

  • ARTEMIS-5647 - Fix branding in webpack dev server
  • ARTEMIS-5658 - Filter no longer fully persist between tabs
  • ARTEMIS-5660 - Console status page no longer refreshes
  • ARTEMIS-5676 - Reduce padding in left web console column
