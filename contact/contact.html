CONTACT US

Contributing

We really want you to get involved in Apache ActiveMQ and welcome all contributions. Contribute by raising a Pull Request via GitHub here:
ActiveMQ Artemis GitHub Page https://github.com/apache/activemq-artemis
ActiveMQ 5 GitHub Page https://github.com/apache/activemq

Chat

You can find many ActiveMQ developers and users in IRC:
irc.freenode.net #apache-activemq logs

Report Issues

ActiveMQ Artemis JIRA https://issues.apache.org/jira/browse/ARTEMIS
ActiveMQ 5 JIRA https://issues.apache.org/jira/browse/AMQ
