Release Notes

Verify the Integrity of Downloads

Verify downloaded files using the PGP signature or SHA512 checksums. See the Apache verification guide.

Download the KEYS file and the .asc signature for your download, then run:

gpg --import KEYS
gpg --verify Apache.NMS-*.asc
