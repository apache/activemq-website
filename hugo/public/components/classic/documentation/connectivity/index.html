Connectivity

Home › Components › Documentation › Connectivity

Apache ActiveMQ Classic supports a large number of Cross Language Clients, a wide range of Protocols, and many Containers.

Protocols

  • AMQP
  • AUTO
  • MQTT
  • OpenWire
  • REST
  • Stomp
  • WebSockets
  • XMPP

Cross Language Clients

  • C Integration
  • Delphi and FreePascal
  • ActiveMQ Classic C++ Clients

Containers

  • Spring Support
  • OSGi Integration
  • Tomcat
  • JBoss Integration
  • Geronimo
  • J2EE
  • Resource Adapter
  • JNDI Support

Bridging

  • JMS to JMS Bridge
  • Ajax
