ActiveMQ-CPP Source

Home › CMS Client › ActiveMQ-CPP Source

The ActiveMQ-CPP client code is maintained in a Git repository.

Web Browsing

https://gitbox.apache.org/repos/asf?p=activemq-cpp.git

Checking Out

git clone https://gitbox.apache.org/repos/asf/activemq-cpp.git

Only project developers can commit to the Git tree via this method.

Branches

The master branch targets the next main release. Bug fixes for the previous major release use a dedicated branch.

To build the code after checking out, see Building.
