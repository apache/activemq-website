### Overview

CMS (C++ Messaging Service) is a JMS-like API for C++ for interfacing with Message Brokers such as Apache ActiveMQ. ActiveMQ-CPP is a client only library — a message broker such as Apache ActiveMQ is still needed for your clients to communicate.

ActiveMQ-CPP supports pluggable transports and wire formats, including OpenWire and STOMP over TCP and SSL, plus a Failover Transport for reliable client operation. It is released under the [Apache 2.0 License](http://www.apache.org/licenses/LICENSE-2.0.html).
