## Description:
Apache ActiveMQ is a popular and powerful open source message-oriented
middleware. Apache ActiveMQ is fast, supports many cross language clients and
protocols, comes with easy to use enterprise integration patterns and many
advanced features while fully supporting JMS 2.0, AMQP 1.0, MQTT, Stomp and
REST.

## Project Activity:

#### ActiveMQ Artemis
 - The 2.29.0 release is almost ready for a vote. It includes a handful of bug-fixes and dependency upgrades. The most notable new feature is that now "connector" definitions added via the management API are durable.

#### ActiveMQ Classic
 - ActiveMQ 5.17.4 released, a maintenance release including important fixes.
 - ActiveMQ 5.18.0 released, with major updates including JMS 2.0 features in preview, and a jakarta-based client to support Spring 6
 - ActiveMQ 5.18.1 release (in vote), with a minor fix for activemq-client-jakarta
 - Planning is underway for supporting Jakarta-based broker, tentatively targeted for 5.19.x
